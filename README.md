## ✨ Features
- Glassmorphism design with animated gradients & smooth scroll reveals
- Hero banner, About, Gallery, Services, Subjects, Why-Choose-Us, animated stat counters, Testimonials, FAQ, Contact & Footer
- 🌙 Dark mode toggle (remembers choice)
- 📞 Tap-to-call, 📧 one-click Gmail compose, 💬 WhatsApp (pre-filled message) + floating WhatsApp button
- Contact form that sends enquiry details straight to WhatsApp
- Sticky navbar, back-to-top, loading screen, fully responsive (mobile/tablet/desktop)
- SEO meta tags + structured data

Plain **HTML + CSS + JavaScript** — no build step, no dependencies.
